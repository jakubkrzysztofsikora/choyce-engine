# Gameplay Loop Skill

## Loop pattern
Build -> Earn -> Upgrade -> Decorate -> Share.

## Constraints
- Session goal within 10-20 minutes.
- Hint system must scaffold (nudge -> clue -> near-solution).
- Rewards should encourage creativity, not grind.
