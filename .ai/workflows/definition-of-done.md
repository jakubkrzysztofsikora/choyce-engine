# Definition of Done (Agent Team)

A task is done only if all checks pass:
1. Meets acceptance criteria in backlog.
2. Keeps hexagonal boundaries (domain isolated from adapters).
3. Includes kid safety + parent control impact review.
4. Includes Ollama tool-calling/error-handling notes for AI-related changes.
5. Includes at least one cross-agent review.
6. Includes tests/checks relevant to the changed scope.
